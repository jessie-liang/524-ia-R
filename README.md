# regexcite.rnliang

<!-- badges: start -->

<!-- badges: end -->

The goal of regexcite.rnliang is to calculate the PDF value of any
normal distributions anywhere

## Installation

You can install the development version of regexcite.rnliang from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("jessie-liang/524-ia-R")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(regexcite.rnliang)
## basic example code
normal_pdf_value(0, 1, 1)
#> [1] 0.2419707
```
